### For development
###### Переходим в директорию с проектом
```bash
cd folder-name
```

###### Установка зависимостей
```bash
npm i
```

###### Запуск сборки приложения и веб-сервера:
```bash
npm serve
```

###### Сборка приложения без минификации: 
```bash
npm run build
```

###### Сборка приложения с минификацией: 
```bash
npm run build:prod
```
